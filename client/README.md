# LocalFix Frontend (Open Source Starter)

This is the frontend template for the LocalFix open-source project, built with React (Vite) and Tailwind CSS.

## Folder Structure

* `src/components/` - Reusable UI components (e.g., Navbar, Footer)
* `src/pages/` - Main page views (e.g., Home, Login, Dashboard)
* `src/layouts/` - Page layouts
* `src/services/` - API calls and external integrations
* `src/utils/` - Helper functions
* `src/assets/` - Static files like images and icons

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`

## Contribution Guidelines

This is a starter template meant to be extended by contributors.
Look for `TODO:` comments in the code to see where features (like API integrations or authentication logic) should be added.
Keep your code clean, minimal, and beginner-friendly!
